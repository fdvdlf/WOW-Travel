# Registro de actualizaciones del sistema

Este archivo registra **solo** actualizaciones hechas al sistema.

## Reglas
- Agrega la actualización **más reciente arriba**.
- Usa fecha en formato `YYYY-MM-DD`.
- Incluye responsable, resumen y detalles clave.
- No registrar cambios menores (solo cambios del sistema).

## Plantilla
```
## YYYY-MM-DD — Responsable
- Resumen: ...
- Cambios: ...
- Notas: ...
```

## 2026-01-25 — Codex
- Resumen: Se crea el archivo de registro de actualizaciones.
- Cambios: Se añade `UPDATE_LOG.md` y se indica en el README.
- Notas: Actualizar este archivo en cada cambio.
